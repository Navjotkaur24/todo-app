# My Todos
A clean and minimal todo app built with React + Vite.

## Features
- Add new tasks
- Mark tasks as complete with checkbox
- Delete tasks with delete/cross button
- Dark theme UI

## Tech Stack
- React
- Vite
- CSS

## Run Locally
npm install 
npm run dev
